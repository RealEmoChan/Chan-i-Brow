# Chan'i Brow 🌐

**Chan'i Brow**, Python ile geliştirilmiş hızlı, hafif ve özelleştirilebilir bir web tarayıcısıdır.
Amaç: performans, gizlilik ve özelleştirme özelliklerini tek bir tarayıcıda birleştirmek.

Bu proje aktif olarak geliştirilmektedir ve her hafta yeni özellikler eklenmektedir.

---

# ✨ Özellikler

### ⚡ Performans

* RAM Limiter (65 MB – 2 GB arası sınırlama)
* Akıllı sekme uyutma sistemi
* Kullanılmayan sekmelerde düşük RAM kullanımı (1–5 MB)
* FPS Limiti (15 / 30 / 60 / 144 / Unlimited / VSync)
* Akıllı video optimizasyonu
* Ultra Lite Mode
* Ağ hızlandırıcı

---

### 🛡️ Gizlilik

* Panic Button (Ctrl + Shift + P)
* Anti-Fingerprint Mode (Ctrl + Shift + F)
* WebRTC koruması
* Tracker engelleyici
* DNS-over-HTTPS desteği
* Geçmiş otomatik silme

---

### 🎨 Özelleştirme

* 8 hazır tema
* Özel tema oluşturma
* Dinamik tema sistemi
* Ana menü arka plan özelleştirme
* Sekme animasyonları

---

### 📂 Sekme Yönetimi

* Sekme gruplama

### 🧰 Araçlar

# 📦 Kurulum

```bash
cd chanibrow
python main.py
```
